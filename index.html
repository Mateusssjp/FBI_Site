ABOUT

The Birth of the FBI Project

The FBI project is developed as part of the PAP and was born with the goal of creating a complete game, combining exploration, tactical combat, progression and narrative.


The game presents a universe inspired by secret operations, advanced technology and conflicts between factions, where the player progresses through missions, upgrades and discoveries.


Each system — weapons, enemies, scenarios, crafting — was designed to be modular, expandable and interconnected, giving the player freedom to evolve in their own way.

The Project's Objective

The game aims to showcase the integration of complex systems: database, mechanics, logic, modular organization and functional design.


The focus is to create an experience where every decision has an impact: managing weapons, completing missions, dealing with enemies, interacting with NPCs and unlocking achievements.


The player grows both in gameplay and in knowledge of the world, using resources, strategies and specific tools to advance.

Systems and Structure

The project uses a structured database with dozens of relationships: weapons, missions, NPCs, enemies, items, blueprints and more.


The system was designed to be scalable, allowing new mechanics to be added without changing the base architecture.


The structure combines table links, game logic, categories, levels and rewards, creating an interactive and expandable world.
